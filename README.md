# ECG Heart Rate Analysis

This project demonstrates a simple Python script to analyze ECG signals and estimate heart rate.

## Features
- Load ECG data from CSV
- Detect R-peaks using SciPy
- Estimate heart rate in bpm
- Plot ECG signal with detected peaks

## Requirements
- pandas
- matplotlib
- scipy

## How to run
1. Install requirements: `pip install -r requirements.txt`
2. Run script: `python ecg_analysis.py`
